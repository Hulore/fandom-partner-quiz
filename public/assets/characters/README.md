Сюда складываются портреты персонажей для экрана результата.

Правила:

- имя файла должно совпадать с `imageKey` персонажа в `app.js`;
- поддерживаются `png`, `jpg`, `jpeg`, `webp`;
- после добавления новых файлов нужно пересобрать `character-images.js`, потому что сайт использует встроенные base64-картинки;
- лучше всего смотрятся вертикальные портреты около `4:5`.

Сейчас в проекте подключены изображения для всех персонажей из теста.
